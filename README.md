# Newton-Fractal

You need to use pip to install numpy and matplotlib.

Specify the interactive resolution (recomended dimension: 100-200), and high quality resolution and the color map (recommended: viridis or twilight).
Consult https://matplotlib.org/stable/users/explain/colors/colormaps.html for a color map that should your fractal porn taste.

Drag the roots around and see how newton methods procude different results.

Press Enter to save the current plot at high resolution.
